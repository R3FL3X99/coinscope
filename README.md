# CoinScope

CoinScope is a modern crypto analytics dashboard built with React and TypeScript.

It allows users to search digital assets, analyze historical price trends, and track coins using a persistent watchlist.

# Tech Stack
- React + TypeScript
- Vite
- React Router
- TanStack React Query
- Recharts
- Tailwind CSS
- CoinGecko API

# Features
- Debounced coin search
- Dynamic timeframe-based price charts (24H / 7D / 30D / 1Y)
- Trend detection (Uptrend / Downtrend)
- All-time high & low metrics
- Community sentiment display
- Persistent watchlist (localStorage)
- Batched API requests
- Fully responsive layout
